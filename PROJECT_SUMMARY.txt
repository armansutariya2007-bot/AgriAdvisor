===============================================
  DAILY USAGE WORKFLOW
===============================================

Step 1: Start XAMPP
   - Open XAMPP Control Panel
   - Click "Start" next to MySQL
   - Wait for green "Running" status

Step 2: Start AgriAdvisor
   - Double-click: START_SERVER.bat
   - Wait 3 seconds
   - Browser opens automatically

Step 3: Login & Use
   - Enter username and password
   - Access dashboard
   - Use all features!

Step 4: Stop (When Done)
   - Close browser
   - Close blue backend window
   - Or press Ctrl+C in window


===============================================
  COMMON TASKS
===============================================

View Database:
  1. Start Apache in XAMPP
  2. Open: http://localhost/phpmyadmin
  3. Click "agriadvisor" database
  4. Browse tables

Configure Email:
  1. Open: backend\.env
  2. Edit SMTP_EMAIL and SMTP_PASSWORD
  3. Use Gmail App Password
  4. Save file

Backup Database:
  1. Open phpMyAdmin
  2. Click "agriadvisor"
  3. Click "Export" tab
  4. Click "Go" to download

Restore Database:
  1. Open phpMyAdmin
  2. Click "agriadvisor"
  3. Click "Import" tab
  4. Choose backup file
  5. Click "Go"


===============================================
  TROUBLESHOOTING QUICK REFERENCE
===============================================

Problem: Python not found
Solution: Reinstall with "Add to PATH" checked

Problem: MySQL won't start
Solution: Check port 3306 is free, restart XAMPP

Problem: Backend won't start
Solution: Run COMPLETE_MYSQL_SETUP.bat again

Problem: Can't login
Solution: Check username/password, register if new

Problem: No OTP email
Solution: Configure Gmail App Password in .env

Problem: Website won't open
Solution: Check backend window is running
